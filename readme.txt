Чтобы это дело развернуть, необходимо:
	1) Залить директорию status на сервер (использовался OpenServer)
	2) Настроить сервер (если не был)
	3) Импортировать базу данных status.sql (у меня не импортировалась - восстанавливал через sql запросы)
	4) Указать свои данные для соединения с базой данных в файле status\php\db.php
	5) В директории status\py в файле run_server.bat указать путь к интерпретатору питон 
	6) Установить на интерпретатор питон все необходимые библиотеки
	7) Помолиться

Чтобы запустить:
	1) Запустить сервер, куда все залито
	2) Запустить сервер нейросетей status\py\run_server.bat
	3) Открыть файл status\index.php в браузере
